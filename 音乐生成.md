# 音乐生成

通过提示参数定义音乐的风格、情绪和场景，通过歌词参数提供演唱的歌词内容。该功能可用于为视频、游戏或应用快速生成独特的背景音乐和主题曲。

## 音乐 2.6：以声传情，翻唱入心，器乐入魂

音乐 2.6 支持更丰富的音乐生成能力，包括：

- **歌曲生成**：根据歌词和音乐风格生成完整歌曲。
- **翻唱生成**：基于已有歌曲生成不同风格的翻唱版本。
- **纯音乐生成**：通过 `is_instrumental` 参数生成无歌词器乐。

## 音乐生成示例

下面以生成一首欢快的新年歌曲为例，演示如何通过两步完成从灵感到成品的完整流程：

1. 调用歌词生成接口，根据主题自动写词。
2. 调用音乐生成接口，根据歌词和风格创作完整歌曲。

如果你已经有歌词，可以跳过第一步；如果没有歌词，也可以直接调用音乐生成接口，并将 `lyrics_optimizer` 参数设置为 `true`。

### 步骤 1：调用歌词生成接口，根据主题生成歌词（可选）

告诉模型你想要的主题，例如“一首欢乐的新年歌曲”。歌词生成接口会自动写出包含主歌、副歌、桥段等结构的完整歌词。

```python
import os
import requests

url = "https://api.minimaxi.com/v1/lyrics_generation"
api_key = os.environ.get("MINIMAX_API_KEY")

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}",
}

payload = {
    "mode": "write_full_song",
    "prompt": "一首欢乐的新年歌曲",
}

response = requests.post(url, json=payload, headers=headers)

print(response.text)
```

### 步骤 2：调用音乐生成接口，谱曲并生成完整歌曲

拿到歌词后，通过 `prompt` 设置音乐风格，例如 `Mandopop, Festive, Upbeat`，并将歌词填入 `lyrics` 参数。音乐生成接口会完成编曲、演唱和音频输出。

相关参数：

- **`lyrics_optimizer`**：没有歌词时，可设置为 `true`，由模型辅助生成歌词。
- **`is_instrumental`**：需要生成纯音乐时，可使用该参数。

参考文档：

- [lyrics_optimizer](https://platform.minimaxi.com/docs/api-reference/music-Generation#body-lyrics-optimizer)
- [is_instrumental](https://platform.minimaxi.com/docs/api-reference/music-Generation#body-is-instrumental)

## 翻唱生成

音乐翻唱可以基于已有歌曲生成不同风格的翻唱版本。支持两种模式：

- **一步翻唱**：直接提供参考音频，系统自动通过 ASR 提取歌词。
- **两步翻唱**：先对音频进行前处理，提取并修改歌词后再生成翻唱。

## 一步翻唱（快捷模式）

将参考音频 URL 和风格描述直接传入音乐生成接口，系统会从音频中自动提取歌词并生成翻唱版本。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_generation"
api_key = os.environ.get("MINIMAX_API_KEY")

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}",
}

payload = {
    "model": "music-cover",
    "audio_url": "https://example.com/original-song.mp3",
    "prompt": "爵士风格，慵懒深夜酒吧，萨克斯",
    "audio_setting": {
        "sample_rate": 44100,
        "bitrate": 256000,
        "format": "mp3",
    },
    "output_format": "url",
}

response = requests.post(url, headers=headers, json=payload)
result = response.json()

print(json.dumps(result, ensure_ascii=False, indent=2))
```

## 两步翻唱（进阶模式，支持歌词修改）

如果需要修改歌词，可以使用两步流程：

1. 调用前处理接口，提取音频特征和格式化歌词。
2. 修改歌词后，调用音乐生成接口生成翻唱版本。

### 步骤 1：处理参考音频

调用[翻唱前处理接口](/api-reference/music-cover-preprocess)提取音频特征和格式化歌词。此步骤**免费**，不会生成音频。

返回内容包括：

- **`cover_feature_id`**：音频特征唯一标识，有效期 24 小时。
- **`formatted_lyrics`**：带段落标签（如 `[Verse]`、`[Chorus]`）的格式化歌词，可自由编辑。
- **`struction_result`**：JSON 字符串，包含模板类型和时间。
- **`audio_duration`**：参考音频时长，单位为秒。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_cover_preprocess"
api_key = os.environ.get("MINIMAX_API_KEY")

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}",
}

payload = {
    "model": "music-cover",
    "audio_url": "https://example.com/original-song.mp3",
}

response = requests.post(url, headers=headers, json=payload)
result = response.json()

cover_feature_id = result["cover_feature_id"]
formatted_lyrics = result["formatted_lyrics"]

print(f"特征 ID: {cover_feature_id}")
print(f"提取歌词:\n{formatted_lyrics}")
```

### 步骤 2：修改歌词并生成翻唱

查看并编辑上一步返回的 `formatted_lyrics`，然后将 `cover_feature_id` 和修改后的歌词传入[音乐生成接口](/api-reference/music-Generation)。

注意事项：

- 使用 `cover_feature_id` 时，不要同时传入 `audio_url` 或 `audio_base64`，三者互斥。
- `lyrics` 参数为必填，长度为 10–1000 个字符。

```python
import json
import os
import requests

url = "https://api.minimaxi.com/v1/music_generation"
api_key = os.environ.get("MINIMAX_API_KEY")

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {api_key}",
}

modified_lyrics = """
[Verse 1]
这里是修改后的第一段歌词
用新的文字讲述你的故事

[Chorus]
全新的副歌部分
用不同的演唱感觉
"""

payload = {
    "model": "music-cover",
    "cover_feature_id": cover_feature_id,
    "lyrics": modified_lyrics,
    "prompt": "爵士风格，慵懒深夜酒吧，萨克斯",
    "audio_setting": {
        "sample_rate": 44100,
        "bitrate": 256000,
        "format": "mp3",
    },
    "output_format": "url",
}

response = requests.post(url, headers=headers, json=payload)
result = response.json()

print(json.dumps(result, ensure_ascii=False, indent=2))
```
